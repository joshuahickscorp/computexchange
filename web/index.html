simplicity

The core is two binaries · a Rust supplier agent and a Go control plane. The job queue is a plain Postgres table claimed with FOR UPDATE SKIP LOCKED · there is no Kafka, RabbitMQ, NATS, or Redis anywhere in the dependency tree. There is no sign-in screen · on first use the console silently provisions a device-local account, and the only credential connected by hand is a payment method. Buyers speak an OpenAI-compatible Batch API at /v1/files and /v1/batches, or a Python SDK that is a single file importing only the standard library.
