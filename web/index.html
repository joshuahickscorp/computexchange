queuethe job queue is a plain Postgres table claimed with FOR UPDATE SKIP LOCKED · no Kafka, RabbitMQ, NATS, or Redis in the dependency tree

general computethe sandbox that runs your own container is built and hardened · no network, read-only root, all capabilities dropped, non-root, a hard timeout · reputation-trusted, not output-verified, and CUDA-only · Linux with Docker and the NVIDIA Container Toolkit · metering it per GPU-second and opening the lane is the road ahead
schedulertasks are sized for a 45 second runtime target · five hard filters gate every claim · quiet hours are enforced by the agent on the device itself
verificationhoneypot known-answer tasks, within-class redundancy, N-way majority votes · a 0.25 audit floor, so even the most trusted supplier is spot-checked about one task in four · a failed honeypot docks reputation, claws back the credit, and quarantines the supplier
quotea quote before any spend at POST /v1/quote · the max_usd cap set at submission is enforced at dispatch, pausing the job before the cap is breached · $0.002 per 1,000 units of Llama 3.2 1B, from $0.004 per audio minute of Whisper

railsconfigured Stripe code is not proof that money moved. Production-money evidence remains an external gate.
