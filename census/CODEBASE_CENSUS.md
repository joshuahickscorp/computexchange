## Layer definitions

- **kernel**: owned, non-test, non-vendor source in {control, agent, spec-engine, token-spec-poc, cli, proto, db} in code languages (go/rust/sql/proto/shader).
- **active_product**: kernel + those dirs' tests + {sdk, web(non-vendor), macapp}.
- **active_repo_support**: scripts/ops/monitoring/config that ship in the default checkout but are not the product itself.
- **historical**: docs (candidate for docs-archive pack).
- **pack:NAME**: relocatable into an optional immutable pack (render-lab, benchmarks, docs-archive, proof-archive).
- **vendored**: under a vendor/ path (upstream, accounted separately).
- **generated**: deterministically regenerable / machine-emitted.
